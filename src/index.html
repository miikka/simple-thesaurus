Simple Thesaurus

This is a thesaurus that suggests only the top 1000 most common English words. Based on a blog post by Morten Just, who thinks that it is a poor idea. (I agree.) Uses Moby Thesaurus and Peter Norvig's word lists that are derived from Google Web Trillion Word Corpus. Implemented in Reason and styled with Tachyons. Built to try out these two technologies.
